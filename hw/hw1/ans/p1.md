# Problem 1 — Regularized MDPs

## 1(a) Contraction of the regularized Bellman expectation operator

The regularized Bellman expectation operator is

$$
[T^\pi_\Omega V](s)
= R^\pi(s) + \Omega(\pi(\cdot|s)) + \gamma \sum_{s'} P^\pi(s'|s) V(s').
$$

To show that it is a contraction under the sup norm, take any two value functions
$V$ and $W$. Then

$$
\begin{aligned}
[T^\pi_\Omega V](s) - [T^\pi_\Omega W](s)
&= \gamma \sum_{s'} P^\pi(s'|s)\bigl(V(s')-W(s')\bigr).
\end{aligned}
$$

The regularizer term cancels because it does not depend on $V$.
Now apply the triangle inequality:

$$
\begin{aligned}
\bigl|[T^\pi_\Omega V](s)-[T^\pi_\Omega W](s)\bigr|
&\le \gamma \sum_{s'} P^\pi(s'|s)\,|V(s')-W(s')| \\
&\le \gamma \|V-W\|_\infty \sum_{s'} P^\pi(s'|s) \\
&= \gamma \|V-W\|_\infty.
\end{aligned}
$$

Taking the maximum over all states $s$ gives

$$
\|T^\pi_\Omega V - T^\pi_\Omega W\|_\infty \le \gamma \|V-W\|_\infty.
$$

Since $0 \le \gamma < 1$, $T^\pi_\Omega$ is a contraction mapping in the
$\ell_\infty$ norm.

## 1(b) Value iteration for the entropy-regularized optimal MDP

With Shannon entropy regularization

$$
\Omega(\pi(\cdot|s)) = H(\pi(\cdot|s))
= -\sum_a \pi(a|s)\ln \pi(a|s),
$$

the regularized optimality backup can be written as

$$
Q^*_\Omega(s,a) = R(s,a) + \gamma \sum_{s'} P(s'|s,a) V^*_\Omega(s')
$$

This follows directly from the given Eq. (3), and

$$
V^*_\Omega(s) = \max_{\pi(\cdot|s)} \left\{
\sum_a \pi(a|s) Q^*_\Omega(s,a) + H(\pi(\cdot|s))
\right\}.
$$

To make the connection with Equation (2) explicit, expand the regularized
statewise objective:

$$
\begin{aligned}
V^*_\Omega(s)
&= \max_{\pi(\cdot|s)}
\left\{
\sum_a \pi(a|s)R(s,a)
+\Omega(\pi(\cdot|s))
+\gamma \sum_a \pi(a|s)\sum_{s'}P(s'|s,a)V^*_\Omega(s')
\right\} \\
&= \max_{\pi(\cdot|s)}
\left\{
\sum_a \pi(a|s)Q^*_\Omega(s,a)
-\sum_a \pi(a|s)\ln \pi(a|s)
\right\}.
\end{aligned}
$$

Now maximize with the constraint $\sum_a \pi(a|s)=1$ using a Lagrange
multiplier. For each action $a$,

$$
\frac{\partial}{\partial \pi(a|s)}
\left[
\sum_b \pi(b|s)Q^*_\Omega(s,b)
-\sum_b \pi(b|s)\ln \pi(b|s)
+\lambda\left(\sum_b \pi(b|s)-1\right)
\right] = 0,
$$

which gives

$$
Q^*_\Omega(s,a) - \ln \pi^*_\Omega(a|s) - 1 + \lambda = 0
\quad \Rightarrow \quad
\pi^*_\Omega(a|s) \propto \exp(Q^*_\Omega(s,a)).
$$

Substituting back yields the closed form

$$
V^*_\Omega(s) = \ln \sum_a \exp(Q^*_\Omega(s,a)).
$$

This is the entropy-regularized Bellman optimality equation for the prompt.
The maximizer is the softmax policy

$$
\pi^*_\Omega(a|s) = \frac{\exp(Q^*_\Omega(s,a))}{\sum_b \exp(Q^*_\Omega(s,b))},
$$

### Pseudocode

```text
Input: states S, actions A, reward model R, transition model P, discount γ, tolerance ε
Initialize V(s) = 0 for all s

repeat
    for each state s in S do
        for each action a in A do
            Q_new(s,a) ← R(s,a) + γ Σ_{s'} P(s'|s,a) V(s')
        end for

        V_new(s) ← ln Σ_a exp(Q_new(s,a))
        π_new(a|s) ← exp(Q_new(s,a) - V_new(s))   # softmax policy
    end for

    δ ← max_s |V_new(s) - V(s)|
    V ← V_new
until δ < ε

for each state s in S do
    for each action a in A do
        Q_star(s,a) ← R(s,a) + γ Σ_{s'} P(s'|s,a) V(s')
    end for
    π_star(a|s) ← exp(Q_star(s,a)) / Σ_b exp(Q_star(s,b))
end for

Output: V*Ω = V, Q*Ω = Q_star, π*Ω = π_star
```

### Line-by-line explanation

1. `Initialize V(s) = 0`: start from an arbitrary value function.
2. `Q_new(s,a) ← ...`: compute the one-step lookahead value for each state-action pair.
3. `V_new(s) ← ln Σ_a exp(Q_new(s,a))`: this is the closed-form solution of Equation (2) after expanding the entropy term and solving the constrained maximization.
4. `π_new(a|s) ← exp(Q_new(s,a) - V_new(s))`: recover the greedy regularized policy; this is exactly a softmax over $Q$.
5. `δ ← max_s |V_new(s) - V(s)|`: measure the largest change across states.
6. `repeat until δ < ε`: because the operator is a contraction, this converges to the unique fixed point.
7. `for each state s in S ...`: recompute the final optimal $Q^*_\Omega$ and $\pi^*_\Omega$ using the converged value function.
8. `Output`: after convergence, the returned values are $V^*_\Omega$, $Q^*_\Omega$, and the corresponding optimal policy.
